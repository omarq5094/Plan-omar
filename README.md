# جدولك

موقع بسيط لإنشاء جدول يومي تلقائيًا من تاريخ بداية إلى تاريخ نهاية.

## المميزات

- إنشاء الأيام تلقائيًا.
- عرض اليوم بالعربية.
- عرض التاريخ الميلادي.
- عرض التاريخ الهجري باستخدام تقويم أم القرى عند دعم المتصفح.
- إضافة أعمدة مخصصة بزر `+`.
- تعديل الخلايا مباشرة.
- حفظ تلقائي في `localStorage`.
- تصدير الجدول كصورة PNG.
- نسخ الجدول.

## طريقة التشغيل

افتح ملف `index.html` مباشرة في المتصفح.

## النشر على GitHub Pages

1. أنشئ Repository جديد في GitHub.
2. ارفع الملفات:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
3. افتح Settings.
4. ادخل Pages.
5. اختر Branch: `main`.
6. اختر Folder: `/root`.
7. اضغط Save.

بعدها يظهر رابط الموقع.


## ملاحظة تصدير الصورة

تم تعديل تصدير الصورة في نسخة v2 ليستخدم مكتبة `html2canvas` عبر CDN، لأن طريقة `foreignObject` قد لا تعمل بشكل ثابت في بعض المتصفحات، خصوصًا مع العربية أو عند فتح الملف مباشرة من الجهاز.

لأفضل نتيجة:
- ارفع المشروع على GitHub Pages.
- أو شغله عبر Live Server داخل VS Code.
