# جدولك

موقع بسيط لإنشاء جدول يومي تلقائيًا من تاريخ بداية إلى تاريخ نهاية.

## المميزات

- إنشاء الأيام تلقائيًا.
- عرض اليوم بالعربية.
- عرض التاريخ الميلادي.
- عرض التاريخ الهجري باستخدام تقويم أم القرى عند دعم المتصفح.
- إضافة أعمدة مخصصة بزر `+`.
- تعديل الخلايا مباشرة.
- حفظ تلقائي في `localStorage`.
- تصدير الجدول كصورة PNG.
- نسخ الجدول.

## طريقة التشغيل

افتح ملف `index.html` مباشرة في المتصفح.

## النشر على GitHub Pages

1. أنشئ Repository جديد في GitHub.
2. ارفع الملفات:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
3. افتح Settings.
4. ادخل Pages.
5. اختر Branch: `main`.
6. اختر Folder: `/root`.
7. اضغط Save.

بعدها يظهر رابط الموقع.

اسم المشروع: جدولك |Planomar

جدولك هو Web App عربي خفيف ومرن لإنشاء الجداول الزمنية اليومية بطريقة ذكية وسريعة، بدون تسجيل دخول معقد وبدون قاعدة بيانات خارجية إلزامية. فكرة المشروع بدأت من احتياج بسيط: إنشاء جدول من تاريخ بداية إلى تاريخ نهاية، ثم إضافة أعمدة مخصصة مثل الخطة، المادة، الحالة، الملاحظات، أو أي عمود يريده المستخدم.

الموقع يولّد الأيام تلقائيًا بناءً على الفترة المحددة، ويعرض اليوم، التاريخ الميلادي، والتاريخ الهجري، مع إمكانية إظهار أو إخفاء أي من هذه الأعمدة. بعد ذلك يستطيع المستخدم تعديل الخلايا مباشرة، إضافة أعمدة جديدة، ترتيب الأعمدة بالسحب، تغيير عرض الأعمدة، وتصدير الجدول كصورة PNG جاهزة للمشاركة.

الهدف من المشروع:
تقديم أداة بسيطة وعملية تساعد الطلاب، الموظفين، صناع المحتوى، وأي شخص يحتاج إلى تنظيم فترة زمنية في جدول واضح وقابل للتخصيص.

الفكرة الأساسية:
يدخل المستخدم إلى الموقع، يحدد تاريخ البداية وتاريخ النهاية، ثم يضغط إنشاء الجدول. يقوم الموقع تلقائيًا بإنشاء صف لكل يوم داخل الفترة، مع اليوم والتاريخ الميلادي والهجري. بعد ذلك يستطيع المستخدم إضافة الأعمدة التي يحتاجها وتعبئتها يدويًا أو باستخدام أدوات مساعدة مثل تعبئة نطاق أو الاقتراحات الجاهزة.

الملفات الأساسية في المشروع:

1. index.html
هذا الملف يحتوي على هيكل الموقع الأساسي. يشمل أقسام الصفحة الرئيسية، منطقة العنوان، أزرار التحكم، الجدول، النوافذ المنبثقة، نافذة أول دخول للمستخدم، نافذة حفظ الجداول، نافذة تعبئة النطاق، وقائمة الاقتراحات.

أهم مكونات index.html:
- Hero Section: يحتوي على اسم المشروع ووصف مختصر وزر تغيير الاسم وزر Dark / Light Mode.
- Controls Section: مقسم إلى مجموعات منظمة:
  - إعداد الجدول.
  - التحرير.
  - الحفظ والإخراج.
- Table Section: يحتوي على الجدول الرئيسي ومؤشر الإنجاز.
- Modals:
  - نافذة سؤال الاسم أول مرة.
  - نافذة جداولي.
  - نافذة تعبئة نطاق.
- Quick Menu:
  - قائمة الاقتراحات المقسمة إلى مجموعات.

2. styles.css
هذا الملف مسؤول عن تصميم الموقع بالكامل. التصميم يعتمد على أسلوب حديث باستخدام Glassmorphism وخلفيات متدرجة، مع دعم كامل للوضع الداكن والوضع الفاتح.

أهم ما يحتويه styles.css:
- متغيرات CSS للألوان والخلفيات.
- تصميم Dark Mode و Light Mode.
- تصميم الأزرار والحقول.
- تصميم الجدول.
- تصميم النوافذ المنبثقة.
- تصميم قائمة الاقتراحات.
- تصميم Bottom Sheet للجوال.
- تصميم مؤشر الإنجاز.
- تصميم التصدير كصورة.
- دعم Responsive Design للجوال.

الهوية البصرية:
التصميم عصري وهادئ، يعتمد على ألوان زرقاء وسماوية مع خلفية داكنة أنيقة في Dark Mode. الواجهة مقسمة إلى بطاقات واضحة، والأزرار مرتبة حسب الوظيفة لتقليل التشويش.

3. app.js
هذا هو الملف الأساسي الذي يحتوي على منطق التطبيق بالكامل. يدير إنشاء الجدول، إضافة الأعمدة، حفظ البيانات، الاقتراحات، الإنجاز، التصدير، وتتبع الزوار.

أهم الوظائف داخل app.js:

- إنشاء الجدول:
يقوم بإنشاء صفوف يومية بين تاريخ البداية والنهاية، ويحسب اليوم والتاريخ الميلادي والهجري.

- إضافة الأعمدة:
يسمح للمستخدم بإضافة أعمدة مخصصة بأي اسم يريده.

- تعديل الخلايا:
كل خلية في الأعمدة المضافة قابلة للتعديل المباشر.

- عرض وإخفاء الأعمدة الأساسية:
يمكن إظهار أو إخفاء:
  - اليوم.
  - التاريخ الميلادي.
  - التاريخ الهجري.

- تغيير عرض الأعمدة:
يمكن للمستخدم سحب رأس العمود لتكبير أو تصغير عرضه.

- ترتيب الأعمدة:
يدعم Drag & Drop لترتيب الأعمدة حسب رغبة المستخدم.

- الاقتراحات:
عند الضغط على خلية، تظهر قائمة اقتراحات مقسمة إلى مجموعات:
  - أساسية.
  - دراسة.
  - حالة.
  - تنظيم.
  - عام.
  - أولوية.

كل مجموعة تعرض خياراتها فقط. مثلًا عند اختيار أولوية تظهر:
  - أولوية عالية.
  - أولوية متوسطة.
  - أولوية منخفضة.
  - عاجل.
  - غير عاجل.

ويمكن تشغيل أو إيقاف الاقتراحات من زر مخصص.

- تعبئة نطاق:
ميزة تسمح بتعبئة مجموعة أيام دفعة واحدة. يختار المستخدم:
  - من تاريخ.
  - إلى تاريخ.
  - العمود.
  - النص.

ثم يتم تعبئة النطاق كاملًا بنقرة واحدة.

- نظام الإنجاز:
ميزة اختيارية يمكن تشغيلها أو إيقافها. عند التفعيل تظهر علامة Check بجانب الخلايا، ويمكن تحديد المهمة كمنجزة بدون تغيير النص الأصلي. يظهر مؤشر إنجاز أعلى الجدول يحسب نسبة المهام المنجزة.

- الحفظ المحلي:
يستخدم localStorage لحفظ:
  - الجدول الحالي.
  - الأعمدة.
  - ترتيب الأعمدة.
  - عرض الأعمدة.
  - الجداول المحفوظة.
  - اسم المستخدم.
  - إعدادات الوضع الداكن أو الفاتح.
  - حالة الاقتراحات.
  - حالة نظام الإنجاز.

- حفظ أكثر من جدول:
يدعم حفظ عدة جداول بأسماء مختلفة داخل المتصفح، مع إمكانية:
  - فتح جدول محفوظ.
  - حذف جدول.
  - نسخ جدول كنسخة مستقلة.

- تصدير الصورة:
يمكن تصدير الجدول كصورة PNG، مع دعم Dark Mode و Light Mode، ويظهر في الصورة:
  - عنوان الجدول.
  - الجدول كاملًا.
  - توقيع: Made with omar al ghrman.

- تتبع الزوار عبر Google Sheets:
أول مرة يدخل المستخدم تظهر نافذة تسأله عن اسمه. يتم حفظ الاسم في localStorage، ثم إرسال بيانات الدخول إلى Google Sheets عبر Google Apps Script.

البيانات المرسلة:
  - name.
  - eventType.
  - userAgent.

أما id و createdAt يتم توليدهما داخل Google Apps Script.

أنواع الأحداث:
  - new_user: عند أول دخول.
  - return_visit: عند العودة لاحقًا.
  - name_change: عند تغيير الاسم.

تمت إضافة حماية بسيطة من السبام بحيث لا يتم تسجيل return_visit أكثر من مرة كل 30 دقيقة من نفس الجهاز.

- تشغيل أو إيقاف الاقتراحات.

الفئة المستهدفة:
- الطلاب لتنظيم الاختبارات والمذاكرة.
- المتدربون لتنظيم الدورات.
- الموظفون لتنظيم المهام.
- صناع المحتوى لتنظيم النشر.
- أي شخص يريد إنشاء جدول زمني سريع ومرن.

نقطة قوة المشروع:
المشروع بسيط في الواجهة لكنه غني بالوظائف. لا يحتاج Login حقيقي، ولا Backend معقد، ويعتمد على localStorage و Google Apps Script كحل خفيف وعملي. هذا يجعله مناسبًا للنشر السريع والتطوير المستمر.

فلسفة التصميم:
الواجهة يجب أن تبقى بسيطة، واضحة، وسريعة. كل ميزة جديدة يجب أن تكون اختيارية حتى لا تزعج المستخدم. لذلك تم جعل الاقتراحات ونظام الإنجاز والوضع الداكن وخيارات الأعمدة قابلة للتحكم.

القيمة العملية:
بدل أن يقوم المستخدم بكتابة الجداول يدويًا في Excel أو Notes أو الصور، يستطيع إنشاء جدول كامل خلال ثوانٍ، تخصيصه، حفظه، ثم تصديره كصورة جاهزة للمشاركة.

اسم المنتج:
جدولك

الشعار المقترح:
أنشئ جدولك خلال ثوانٍ.

وصف قصير:
جدولك هو مولّد جداول عربي ذكي ومرن يساعدك على إنشاء جداول زمنية قابلة للتخصيص، مع دعم التاريخ الميلادي والهجري، التصدير كصورة، الحفظ المحلي، والاقتراحات الجاهزة.

وصف تسويقي:
مع جدولك، لا تحتاج إلى إعادة بناء جدولك يدويًا كل مرة. اختر الفترة، أضف أعمدتك، عبّئ خطتك، واحفظها أو شاركها كصورة أنيقة خلال ثوانٍ.

الحالة الحالية:
المشروع في مرحلة MVP متقدمة، جاهز للنشر على GitHub Pages، وقابل للتوسع مستقبلًا بإضافة قوالب جاهزة، تصدير واستيراد JSON، ألوان تلقائية، وإحصائيات استخدام.
